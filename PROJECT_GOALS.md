# Project Goals & Vision

This document continuously tracks the high-level goals, aesthetic choices, and current priorities for this portfolio website. We will use this to ensure every code change aligns with the master vision.

## 🎯 What is the Goal?
I want to create a portfolio website that showcases my skills and experience as a software engineer. I want it to be modern, interactive, and easy to navigate. I also want it to be a place where I can share my thoughts and ideas on technology and software development.


## 🚀 What are we trying to achieve?
I want visitors to feel impressed by my skills and experience. I want them to be able to easily find the information they are looking for and to be able to contact me if they are interested in working with me. I also want them to be able to learn more about me and my interests.


## 🎨 Theme & Aesthetic
I want the website to be modern, interactive, and easy to navigate. I also want it to be a place where I can share my thoughts and ideas on technology and software development. The theme should be minimalist and zen, soft and calm design


## 🌈 Color Palette
Zen color palettes focus on muted, natural tones that encourage tranquility and mindfulness. Drawing inspiration from nature, they favor subtle hues that create harmony without overwhelming the senses. Soft beiges, grays, greens, and pale blues slow heart rates and reduce stress, aligning with Zen principles of simplicity.


## ❓ Functional Requirements
1) I want my project to be sync to my github's repoitories, however i also want to have the option to edit every part of the description, title, and other information about the project.
2) I want to have access to edit the website, add new projects, and update my information. 
3) i want my experience to be sync to my linkedin profile, however i also want to have the option to edit every part of the description, title, and other information about the experience.

## 💻 Tech Stack
Based on the current repository and our architectural decisions, this is your stack:

### Core Framework
- **Framework**: Next.js 16.1 (App Router)
- **Language**: TypeScript 5.7 & React 19

### Styling & UI
- **CSS Framework**: Tailwind CSS 4.2
- **UI Components**: Radix UI Primitives (shadcn/ui style components)
- **Animations**: Framer Motion 11
- **Icons**: Lucide React

### Data Architecture (Planned)
- **Headless CMS**: Sanity.io (For live editing on a visual admin dashboard)
- **API Fetching**: Sanity GROQ Queries 

### Deployment
- **Hosting**: Vercel (or Netlify)
- **Analytics**: Vercel Analytics
